Mysql Group By与临时表的关系

2021-01-25
  1. 如果Group By的列没有索引，产生临时表
  2. 如果Group By时，Select的列不止Group By列一个，并且Group By的列不是主键，产生临时表
  3. 如果Group By的列有索引，Order By的列没有索引，产生临时表
  4. 如果Group By的列和Order By的列不一样，即使都一样都会产生临时表
  5. 如果Group By或者Order By的列来不是自Join语句的第一个表，会产生临时表
  6. 如果Distinct和Group By的列没有索引，产生临时表

内联查询表查询一般的执行过程

  1. 执行FROM语句
  2. 执行ON过滤
  3. 添加外部行
  4. 执行WHERE条件
  5. 执行GROUP BY 分组语句
  6. 执行HAVING
  7. SELECT 列表
  8. 执行DISTINCT去重复数据
  9. 执行ORDER BY语句
  10. 执行Limit子句
